# Entorno DEV

## Probar en vivo
```bash
mvn quarkus:dev
```

## Compilar y ejecutar
```bash
clear
mvn clean package -Dquarkus.profile=dev
java -jar ./target/quarkus-app/quarkus-run.jar 
```
